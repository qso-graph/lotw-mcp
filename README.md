# lotw-mcp

MCP server for [ARRL Logbook of The World](https://lotw.arrl.org/) -- QSO query, QSL status, DXCC credit.

Part of the [qso-graph](https://qso-graph.io/) collection of amateur radio MCP servers.

## Planned Tools

| Tool | Description |
|------|-------------|
| `lotw_query` | Query QSOs from LoTW with date/band/mode filters |
| `lotw_qsl_status` | Check QSL confirmation status for specific QSOs |
| `lotw_dxcc_credit` | DXCC credit summary from LoTW confirmations |
| `lotw_last_upload` | Check when a callsign last uploaded to LoTW |
